# 🇮🇳 Election Mitra AI (v3.0)

An interactive, bilingual AI assistant designed to simplify the election process for all citizens, from children to the elderly.

## 🚀 Key Features
- **🤖 Gemini AI Integration:** Get instant insights on election-related queries.
- **📅 Interactive Timeline:** Step-by-step guidance from Voter List to Results.
- **🌍 Bilingual Support:** Seamless experience in both English and Hindi.
- **♿ Accessibility:** Easy-to-use CLI with visual emoji indicators.

## 🛠️ Technical Stack
- **Language:** Python 3.x
- **Services:** Simulated Google Gemini 1.5 Flash on Google Cloud.
- **Testing:** Automated unit tests included in `test_main.py`.

## 📖 How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python MAIN.PY`
